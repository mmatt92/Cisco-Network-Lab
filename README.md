<h1>Cisco Packet Tracer Network Lab</h1>



<h2>Description</h2>

In this basic lab I configure a virtual network using Cisco Packet Tracer. This will demonstrate how PCs from different networks talk to one another through switches and routers.
<br />


<h2>Utilities Used</h2>
 
- <b>Cisco Packet Tracer</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
